# The MiSTer Manuals DB - Amstrad GX4000
This is a database for the MiSTer project that downloads the English manuals in .pdf form for the Amstrad GX4000 to the docs folder, which can be loaded from the OSD within th core by selecting "Help".

To use it simply copy and paste the below to the bottom of your downloader.ini file (found at: /media/fat/downloader.ini )

Or download this .ini file and put it in /media/fat https://github.com/ajgowans/manualsdb-gx4000/blob/db/downloader_ajgowans_manualsdb-gx4000.ini


```ini
[ajgowans/manualsdb-gx4000]
db_url = https://raw.githubusercontent.com/ajgowans/manualsdb-gx4000/db/db.json.zip
